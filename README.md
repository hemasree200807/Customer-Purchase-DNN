# 🛒 Customer Purchase Prediction System

A machine learning-based Customer Purchase Prediction System developed using **Python**. The application predicts whether a customer is likely to make a purchase based on customer information such as income, age, and spending habits.

---

## 📖 Project Overview

The Customer Purchase Prediction System analyzes customer attributes and predicts purchasing behavior. By evaluating factors such as income, age, and spending habits, the application classifies whether a customer is likely to make a purchase.

This project demonstrates how predictive analytics can assist businesses in understanding customer behavior and supporting data-driven marketing decisions.

---

## ✨ Features

- 🛒 Predict customer purchase decisions
- 💰 Income-based analysis
- 👤 Age-based prediction
- 📊 Spending habit evaluation
- ⚡ Fast prediction results
- 💻 Simple command-line interface
- 🤖 Machine Learning / Deep Learning-based prediction

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/hemasree200807/Customer-Purchase-Prediction.git
```

### Navigate to the project folder

```bash
cd Customer-Purchase-Prediction
```

### Install the required packages

```bash

```

### Run the application

```bash
python app.py
```

---

## ▶️ How It Works

1. Run the application.
2. Enter the customer details:
   - Income
   - Age
   - Spending Habit (1–10)
3. The application processes the input data.
4. The trained prediction model analyzes the customer profile.
5. The system predicts whether the customer is likely to make a purchase.

---

## 📊 Input Parameters

| Parameter | Description |
|-----------|-------------|
| Income | Customer's monthly or annual income |
| Age | Customer's age |
| Spending Habit | Spending behavior represented on a scale from **1 to 10** |

---
